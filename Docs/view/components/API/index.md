### Options
| 配置 | 描述 | 类型 | 接受的值 | 默认值 |
|---------|------------ |---------- |-------------  |-------- |
| type | 选择器的类型 | string | date/date-range | date |
| format | 选择器输出日期的格式 | string | - | yyyy/mm/dd |
| placement | 选择器位于输入框的位置 | string | top/bottom/right/left | bottom |
| disabled | 禁用日期(具体用法见于例子) | function | - | - |

### Event
| 事件名 | 描述  | 
|---------|---------- |
| change | 日期改变时触发的事件  |
