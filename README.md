## flex-date-picker
A datepicker for PC

## Installing

Using npm:

```bash
$ npm install flex-date-picker
```

Using yarn:

```bash
$ yarn add flex-date-picker
```

## Example

```html
<input id=input>
```
```js
import dp from 'flex-date-picker'
(this.$refs.test2.$el,{type:'date-range'})
```
